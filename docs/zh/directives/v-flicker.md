# v-filcker

## 功能介绍

元素闪烁

## 示例

<div v-flicker
  :style="{
    width: '200px',
    height: '200px',
    borderRadius: '5px',
    background: 'blue',
  }"></div>

## 使用

```typescript {3}
<template>
  <div
    v-flicker
    :style="{
      width: '100px',
      height: '100px',
      borderRadius: '5px',
      background: 'blue',
    }"></div>
</template>
```
