# v-draggable

## 功能介绍

使元素可拖拽

## 示例

<script setup lang="ts"></script>
<div
  v-draggable
  :style="{
    width: '100px',
    height: '100px',
    background: 'blue',
    borderRadius: '50px',
    display: 'flex',
    justifyContent: 'center',
    alignItems: 'center',
    transform: 'translateX(50px)',
    color: 'white'
  }"
>
  拖拽我
</div>
