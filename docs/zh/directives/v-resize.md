# v-resize

## 功能介绍

响应缩放元素大小

## 示例

<div
    v-resize
    :style="{
        width: '100px',
        height: '100px',
        background: 'blue',
        borderRadius: '5px'
    }"
  ></div>

## 使用

```typescript {3}
<template>
  <div
    v-resize
    :style="{
        width: '100px',
        height: '100px',
        background: 'blue',
        borderRadius: '5px'
    }"
  ></div>
</template>
```
